UUID Function

Generates a universally unique identifier (UUID) in the standard "version 4" format specified by RFC 4122. Each call to UUID() returns a new unique identifier.

Syntax

UUID()

Return Value

A unique 36-character string like "a5fc1f61-b034-4512-8a9f-f3f30841d5d8".

Example

CREATE TABLE items (id TEXT PRIMARY KEY, name TEXT);
INSERT INTO items VALUES (UUID(), 'Foo'), (UUID(), 'Bar');
