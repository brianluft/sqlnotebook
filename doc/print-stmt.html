PRINT Statement

Prints the expression to the script or console output.

Syntax

Arguments

  • expression (any expression)
    The value to be printed.

Examples

  • PRINT 5;
    Prints 5.
  • PRINT 1 + 2;
    Prints 3.
  • PRINT 'Hello world';
    Prints "Hello world".
  • PRINT (SELECT COUNT(*) FROM mytable);
    Prints the number of rows in mytable.
