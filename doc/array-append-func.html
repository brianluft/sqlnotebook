ARRAY_APPEND Function

Creates a new array by copying an existing array and appending one or more elements to the end. The existing array is not modified.

Syntax

ARRAY_APPEND(array, value1, [value2], ...)

Arguments

  • array (blob expression)
    The existing array to copy.
  • value1, value2, ... (expression)
    The values to append to the array.

Return Value

The new array, with the new values appended to the end.

Examples

  • DECLARE @x = ARRAY_APPEND(ARRAY(1, 2, 3), 99, 100)
    Assigns a 5-element array containing 1, 2, 3, 99, 100 to @x.
